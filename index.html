参考命令

安装

wget http://res.szlt.net/master/lnmp.zip -P /tmp
cd /usr/local/src
unzip /tmp/lnmp.zip
chmod -R 777 sh && cd sh
./install.sh 2>&1 | tee install.log

卸载

cd /usr/local/src
./uninstall.sh 2>&1 | tee uninstall.log

重新安装

cd /usr/local/src
rm -rf sh
rm /tmp/lnmp.zip
wget http://res.szlt.net/master/lnmp.zip -P /tmp
unzip /tmp/lnmp.zip
chmod -R 777 sh && cd sh
./install.sh 2>&1 | tee install.log

关于这个产品

我们满怀对 Flarum 的热爱之情，对其运行环境进行深入探索，并持续对本环境包进行深度优化，旨在使其能够以更优雅的姿势，为 Flarum 提供稳健的运行环境 从此，我们再也不用为无法正常安装 Flarum 或报错连天的环境问题而影响正常使用 此外，为了满足国内外不同区域服务器的快速部署 Flarum 生产环境，我们还启用了三个不同区域的 CDN 服务，分别是位于美国的亚马逊 S3 存储，位于中国的新浪 SAE 和阿里云的 CDN 悉心打造，致臻品质，只为您而来！
